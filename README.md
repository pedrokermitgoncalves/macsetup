# Mac-Setup

Personal Mac setup

Setups personal settings, apps and configs for first time usage

Run in order:

```bash
$ sh presetup.sh
$ sh applications.sh
$ sh macos-preferences.sh
$ sh gitconfig.sh
$ sh fonts.sh
$ sh npm-packages.sh
$ sh vscode-setup.sh # turn on vscode sync settings to avoid this one
$ sh zsh.sh
```
